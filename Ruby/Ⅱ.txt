○ 配列の繰り返し処理

	配列名.each do |変数名|
		変数名が使えるスコープ
	end

○ ハッシュとは

	{キー1 => 値1, キー2 => 値2 }
	値の出力例
		puts ハッシュ名[キー]
	キーと値は追加できる

○ ハッシュの書き方

	キーの先頭に「:」をつける
	{:キー1 => 値1, :キー2 => 値2 }
	「:name」という書き方のことをシンボルと言います
	シンボルは、「""」「''」の代わりに「:」
	
○ シンボルはハッシュを省略できる
	
	befone {:name => "Suzuki", :age => 21 }
	after { name: "suzuki", age: 21 } 

○ nilとは
	
	何もない状態のこと
	
○ nilを利用したif文

	値がある→true	nil→false
